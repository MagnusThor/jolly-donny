Class IndexedDBProvider

Interface representing an offline storage provider. This interface defines methods for initializing, saving, updating, deleting, and querying data in an offline storage system.

        pickKeys?: K[],
    ): Promise<Pick<T, K>[]>

    Finds and retrieves items from an IndexedDB object store based on a query function. Optionally, specific keys can be picked from the matched items.

    Type Parameters

    • T extends PersistedEntityBase

      The type of the persisted entity.

    • K extends string | number | symbol = keyof T

      The keys of the entity to pick (defaults to all keys of T).

    Parameters

    • label: string

      A label or identifier for the operation (not used in the current implementation).

    • query: (item: T) => boolean

      A function that determines whether an item matches the search criteria.

    • OptionalpickKeys: K[]

      An optional array of keys to pick from the matched items. If not provided, the entire item is returned.

    Returns Promise<Pick<T, K>[]>

    A promise that resolves to an array of items matching the query. If pickKeys is provided, the items will only include the specified keys.

    Throws

    Will throw an error if there is an issue accessing the IndexedDB object store.
